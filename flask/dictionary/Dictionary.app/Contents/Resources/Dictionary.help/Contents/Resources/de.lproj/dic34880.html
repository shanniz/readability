Lexikon-Hilfe

Schlagen Sie mit der App „Lexikon“ Definitionen für Wörter und Satzteile in verschiedenen Quellen nach.

Tipp: Sie können außerdem schnell Wörter nachschlagen, während Sie in einer App arbeiten oder im Internet surfen.

Das Lexikon kann direkt vom Launchpad aus geöffnet werden (klicken Sie im Dock auf das Launchpad-Symbol ).

App „Lexikon“ öffnen

Suchen nach einem Wort oder einer Formulierung im Lexikon

  • Geben Sie ein Wort oder eine Formulierung im Suchfeld oben rechts im Lexikon-Fenster ein.

    Hinweis: Wenn Sie eine andere Lexikonquelle hinzufügen, warten Sie, bis der Download beendet wurde, bevor Sie Wörter oder Formulierungen suchen. Weitere Informationen zum Hinzufügen von Quellen finden Sie unter „Anpassen der Quellen für das Lexikon“ weiter unten.

    Wenn Sie nach einem Wort in einer Definition suchen, wird im Suchfeld eine SnapBack-Taste angezeigt. Durch Klicken auf diese Taste gelangen Sie zur Ausgangsdefinition zurück. Sie können auch die Pfeiltasten in der Symbolleiste des Lexikon-Fensters zum Wechseln zwischen den angezeigten Definitionen verwenden.

Vergrößern bzw. Verkleinern von Text

  • Klicken Sie hierzu beliebig oft auf die Tasten mit dem „A“ im Lexikon-Fenster.

    Die neue Schriftgröße bleibt so lange aktiv, bis Sie erneut auf eine der „A“-Tasten klicken.

Anpassen der Quellen für das Lexikon

  • Wählen Sie „Lexikon“ > „Einstellungen“ in der App „Lexikon“ aus. Sie können Quellen wie spanische oder koreanische Wörterbücher auswählen und neu anordnen. Sie können Optionen für eine Quelle festlegen zum Beispiel, wie die Aussprache angezeigt werden soll, oder welche Wikipedia-Sprachversionen durchsucht werden sollen. Die hier ausgewählten Quellen bestimmen, welche Quellen beim Nachschlagen von Wörtern im Lexikon angezeigt werden.

Ausblenden von Obszönitäten

  • Wenn Sie der Administrator eines Mac-Computers sind, können Sie den Zugriff der Benutzer auf obszöne Wörter steuern. Hierzu können Sie auf dem Mac die Kindersicherung einrichten und „Anstößige Ausdrücke im Lexikon ausblenden“ im Bereich „Andere“ der Systemeinstellung „Kindersicherung“ auswählen. Anschließend kann nur ein Administrator die Kindersicherung deaktivieren.

Wenn die Suche nach einem Wort im Lexikon zu keinem Ergebnis führt, ist das Wort möglicherweise nicht in den Lexikon-Einstellungen ausgewählten Quellen vorhanden oder die Kindersicherung wurde aktiviert. Sofern möglich, schlägt die App „Lexikon“ alternative Wörter vor.

Mithilfe von Spotlight können Sie ebenfalls schnell nach einer Definition suchen. Klicken Sie oben rechts auf dem Bildschirm auf das Lupen-Symbol und geben Sie ein Wort bzw. eine Formulierung ein.
