Βοήθεια για το Λεξικό

Χρησιμοποιήστε την εφαρμογή «Λεξικό» για να δείτε ορισμούς λέξεων και φράσεων από διάφορες πηγές.

Συμβουλή: Μπορείτε επίσης να βρείτε γρήγορα λέξεις ενώ χρησιμοποιείτε μια εφαρμογή ή επισκέπτεστε ιστοσελίδες.

Μπορείτε να ανοίξετε το Λεξικό από το Launchpad (κάντε κλικ στο εικονίδιο Launchpad στο Dock).

Άνοιγμα του Λεξικού

Αναζήτηση για λέξη ή φράση στο Λεξικό

  • Πληκτρολογήστε μια λέξη ή φράση στο πεδίο αναζήτησης στην πάνω δεξιά γωνία του παραθύρου Λεξικού.

    Σημείωση: Εάν προσθέσετε μια άλλη πηγή στο Λεξικό, περιμένετε έως ότου ολοκληρωθεί πλήρως η λήψη προτού πραγματοποιήσετε αναζήτηση για λέξη ή φράση. Για πληροφορίες σχετικά με την προσθήκη πηγών, ανατρέξτε στην ενότητα «Προσαρμογή πηγών Λεξικού» παρακάτω.

    Κατά την εύρεση μιας λέξης σε έναν ορισμό, ένα κουμπί SnapBack εμφανίζεται στο πεδίο αναζήτησης. Κάντε κλικ σε αυτό για να επιστρέψετε στον αρχικό ορισμό. Μπορείτε επίσης να χρησιμοποιήσετε τα κουμπιά βέλους στη γραμμή εργαλείων του παραθύρου του Λεξικού για μετακίνηση μεταξύ των ορισμών που έχετε προβάλει.

Μεγέθυνση ή σμίκρυνση κειμένου

  • Κάντε κλικ στα κουμπιά «Α» στη γραμμή εργαλείων του παραθύρου του Λεξικού όσες φορές χρειαστεί.

    Η γραμματοσειρά παραμένει στο ίδιο μέγεθος έως ότου να κάνετε ξανά κλικ στο κουμπί «Α».

Προσαρμογή πηγών Λεξικού

  • Στο Λεξικό, επιλέξτε «Λεξικό» > «Προτιμήσεις». Μπορείτε να επιλέξετε και να αναδιατάξετε πηγές, όπως τα λεξικά Ισπανικών ή Κορεατικών. Μπορείτε να ορίσετε επιλογές για μια πηγή, όπως για παράδειγμα, τον τρόπο εμφάνισης προφορών ή την έκδοση της Βικιπαίδειας σε συγκεκριμένη γλώσσα για αναζήτηση. Οι πηγές που θα επιλέξετε θα καθορίσουν τις πηγές που θα εμφανίζονται στο Λεξικό όταν ψάχνετε λέξεις.

Απόκρυψη αισχρολογιών

  • Εάν είστε ο διαχειριστής ενός Mac, μπορείτε να ελέγχετε την πρόσβαση των άλλων χρηστών για ακατάλληλες λέξεις. Απλώς διαμορφώστε τον γονικό έλεγχο στο Mac και μετά επιλέξτε το πλαίσιο επιλογής «Απόκρυψη αισχρολογίας στο Λεξικό» στο τμήμα «Άλλο» στις προτιμήσεις Γονικού ελέγχου. Μετά τη διαμόρφωση του γονικού ελέγχου, μόνο ο διαχειριστής μπορεί να τον αφαιρέσει.

Εάν εκτελείτε αναζήτηση για μια λέξη και το Λεξικό δεν μπορεί να εντοπίσει κανένα αποτέλεσμα, ενδέχεται η λέξη να μην βρίσκεται στις πηγές που είναι επιλεγμένες στις προτιμήσεις του Λεξικού ή να έχει οριστεί γονικός έλεγχος. Εάν είναι δυνατόν, το Λεξικό προτείνει εναλλακτικές λέξεις.

Μπορείτε επίσης να χρησιμοποιήσετε το Spotlight για να λάβετε έναν γρήγορο ορισμό. Κάντε κλικ στο εικονίδιο του μεγεθυντικού φακού στην επάνω δεξιά γωνία της οθόνης και μετά πληκτρολογήστε μια λέξη ή φράση.
