Aide Dictionnaire

Utilisez l’app Dictionnaire pour afficher des définitions de mots et expressions provenant de différentes sources.

Conseil : Vous pouvez également rechercher rapidement des mots pendant que vous travaillez dans une app ou que vous parcourez des pages web.

Vous pouvez ouvrir Dictionnaire à partir de Launchpad (cliquez sur l’icône Launchpad  dans le Dock).

Ouvrir Dictionnaire

Rechercher un mot ou une expression dans Dictionnaire

  • Saisissez un mot ou une expression dans le champ de recherche situé dans le coin supérieur droit de la fenêtre Dictionnaire.

    Remarque : Si vous ajoutez une autre source de Dictionnaire, attendez qu’elle se télécharge complètement avant de rechercher un mot ou une expression. Pour en savoir plus sur l’ajout de sources, consultez la rubrique « Personnaliser les sources de Dictionnaire » ci-dessous.

    Si vous recherchez un mot dans une définition, un bouton SnapBack apparaît dans le champ de recherche ; cliquez dessus pour revenir à la définition de départ. Les boutons de flèches de la barre d’outils de la fenêtre Dictionnaire servent également à naviguer entre les définitions.

Agrandir ou réduire le texte

  • Cliquez sur les boutons « A » dans la barre d’outils de la fenêtre Dictionnaire autant de fois que nécessaire.

    La police reste ainsi jusqu’à ce que vous cliquiez à nouveau sur ce bouton.

Personnaliser les sources de Dictionnaire

  • Dans Dictionnaire, choisissez Dictionnaire > Préférences. Sélectionnez les sources (telles que les dictionnaires espagnol et coréen) et réorganisez-les. Vous pouvez définir des options pour une source, telles que le mode d’affichage de la phonétique ou la langue à rechercher sur Wikipédia. Les sources que vous sélectionnez ici sont utilisées dans Dictionnaire et lorsque vous recherchez des mots.

Masquer les grossièretés

  • Si vous êtes administrateur sur un Mac, vous pouvez contrôler l’accès des autres utilisateurs aux mots inconvenants. Il vous suffit d’activer le contrôle parental sur le Mac, puis de cocher la case Masquer les grossièretés dans le dictionnaire dans la sous-fenêtre Autre des préférences du Contrôle parental. Une fois cette opération effectuée, seul un administrateur peut désactiver le contrôle parental.

Si vous recherchez un mot et que Dictionnaire ne trouve aucun résultat, peut-être que celui-ci n’existe pas dans les sources sélectionnées dans les préférences de Dictionnaire, ou que le contrôle parental est activé. Si possible, Dictionnaire suggère des mots proches.

Vous pouvez également utiliser Spotlight pour trouver une définition rapidement. Cliquez sur l’icône de loupe  dans le coin supérieur droit de l’écran, puis saisissez un mot ou une expression.
